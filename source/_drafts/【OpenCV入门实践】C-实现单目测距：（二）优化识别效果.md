# 降噪

书接上文。我们之前虽然已经实现了对画面中圆形的识别，但其效果并不能令人满意。一个重要的原因是，霍夫圆检测是**噪声敏感**的。画面中的噪声容易对圆的检测造成不小的干扰，因此我们将首先将图像降噪。
